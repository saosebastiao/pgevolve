# Types (Enums, Domains, Composites) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring `CREATE TYPE AS ENUM`, `CREATE DOMAIN`, and `CREATE TYPE AS (...)` under pgevolve's managed surface — full lifecycle (CREATE/ALTER/DROP) with cascade drop+recreate for changes PG itself cannot ALTER.

**Architecture:** A single new `UserType { qname, kind: UserTypeKind, comment }` IR record on `Catalog.types`, where `UserTypeKind` discriminates `Enum | Domain | Composite`. The existing `ColumnType::UserDefined(QualifiedName)` is the column-side reference; this plan adds the AST-resolution check that wires the two sides together. Differ produces a single `UserTypeChange` enum with one variant per supported ALTER op plus a `ReplaceWithCascade` fallback. Planner emits 12 new step kinds; dependent-recreation walker (from v0.2 views) gets extended to cover type→column / type→composite / type→domain / type→view edges.

**Tech Stack:** Rust 1.78+; pg_query 6.x for source parsing; tokio_postgres for catalog read; the existing `NormalizedExpr` (v0.1), `NormalizedBody` (v0.2-readiness), `DepEdge`/`DepSource` (v0.2-readiness), and AST-resolution machinery (v0.2-readiness).

**Source spec:** [`docs/superpowers/specs/2026-05-18-types-design.md`](../specs/2026-05-18-types-design.md)

```rust
use crate::ir::column_type::ColumnType;
use crate::parse::normalize_expr::NormalizedExpr;

#[derive(Clone, Debug, Eq, PartialEq, Hash, Serialize, Deserialize)]

    pub qname: QualifiedName,
    pub kind: UserTypeKind,
    pub comment: Option<String>,
}

#[derive(Clone, Debug, Eq, PartialEq, Hash, Serialize, Deserialize)]
#[serde(tag = "kind", rename_all = "snake_case")]
pub enum UserTypeKind {
    Enum { values: Vec<EnumValue> },
    Domain {
        base: ColumnType,
        nullable: bool,
        default: Option<NormalizedExpr>,
        check_constraints: Vec<DomainCheck>,
        collation: Option<QualifiedName>,
    },
    Composite { attributes: Vec<CompositeAttribute> },
}

#[derive(Clone, Debug, PartialEq, Serialize, Deserialize)]
pub struct EnumValue {
    pub name: String,
    /// PG's `pg_enum.enumsortorder` is real4; we store the same float for
    /// byte-stable round-trip.
    pub sort_order: f32,
}

// f32 doesn't implement Eq or Hash. Provide manual impls that use the
// IEEE 754 bit pattern, so two EnumValues with the same sort_order
// compare equal and hash equal even though f32: !Eq in std.
impl Eq for EnumValue {}
impl std::hash::Hash for EnumValue {
    fn hash<H: std::hash::Hasher>(&self, state: &mut H) {
        self.name.hash(state);
        self.sort_order.to_bits().hash(state);
    }
}

#[derive(Clone, Debug, Eq, PartialEq, Hash, Serialize, Deserialize)]
pub struct DomainCheck {
    pub name: Identifier,
    pub expression: NormalizedExpr,
}

pub struct CompositeAttribute {
    pub name: Identifier,
    pub ty: ColumnType,
    pub collation: Option<QualifiedName>,
}
```

```rust
//! Source-side parser for `CREATE TYPE x AS ENUM (...)`.

use pg_query::protobuf::CreateEnumStmt;

use crate::identifier::{Identifier, QualifiedName};
use crate::ir::user_type::{EnumValue, UserType, UserTypeKind};
use crate::parse::builder::shared::{
    qname_from_qualified_list, str_from_string_node,
};
use crate::parse::error::{ParseError, SourceLocation};

pub(crate) fn build_enum(
    stmt: &CreateEnumStmt,
    file: &std::path::Path,
    location: SourceLocation,
) -> Result<UserType, ParseError> {
    // stmt.type_name is a list of String nodes: [schema, name] or [name].
    let qname: QualifiedName = qname_from_qualified_list(
        &stmt.type_name,
        file,
        location.clone(),
    )?;

    // stmt.vals is a list of String nodes — the enum value labels.
    if stmt.vals.is_empty() {
        return Err(ParseError::Structural {
            location,
            message: format!("CREATE TYPE {qname} AS ENUM requires at least one value"),
        });
    }

    let mut values = Vec::with_capacity(stmt.vals.len());
    let mut seen = std::collections::BTreeSet::new();
    for (idx, node) in stmt.vals.iter().enumerate() {
        let label = str_from_string_node(node).ok_or_else(|| ParseError::Structural {
            location: location.clone(),
            message: format!(
                "CREATE TYPE {qname} AS ENUM: value at position {idx} is not a string literal"
            ),
        })?;
        if !seen.insert(label.clone()) {
            return Err(ParseError::Structural {
                location: location.clone(),
                message: format!(
                    "CREATE TYPE {qname} AS ENUM: duplicate value {label:?}"
                ),
            });
        }
        // 1-indexed floats: 1.0, 2.0, 3.0, ...
        values.push(EnumValue {
            name: label,
            #[allow(clippy::cast_precision_loss)]
            sort_order: (idx as f32) + 1.0,
        });
    }

    let _ = file; // location already carries it; suppress unused if not used otherwise
    Ok(UserType {
        qname,
        kind: UserTypeKind::Enum { values },
        comment: None,
    })
}
```

```rust
//! Source-side parser for `CREATE DOMAIN`.

use pg_query::protobuf::{ConstrType, CreateDomainStmt};

use crate::identifier::{Identifier, QualifiedName};
use crate::ir::column_type::ColumnType;
use crate::ir::user_type::{DomainCheck, UserType, UserTypeKind};
use crate::parse::builder::shared::{
    column_type_from_type_name, qname_from_qualified_list,
};
use crate::parse::error::{ParseError, SourceLocation};
use crate::parse::normalize_expr::NormalizedExpr;

pub(crate) fn build_domain(
    stmt: &CreateDomainStmt,
    file: &std::path::Path,
    location: SourceLocation,
) -> Result<UserType, ParseError> {
    let qname = qname_from_qualified_list(&stmt.domainname, file, location.clone())?;

    let type_name = stmt.type_name.as_ref().ok_or_else(|| ParseError::Structural {
        location: location.clone(),
        message: format!("CREATE DOMAIN {qname}: missing base type"),
    })?;
    let base: ColumnType = column_type_from_type_name(type_name).map_err(|e| ParseError::Structural {
        location: location.clone(),
        message: format!("CREATE DOMAIN {qname}: unsupported base type — {e}"),
    })?;

    let mut nullable = true;
    let mut default: Option<NormalizedExpr> = None;
    let mut checks: Vec<DomainCheck> = Vec::new();
    let mut seen_check_names = std::collections::BTreeSet::new();

    for con in &stmt.constraints {
        let Some(node) = &con.node else { continue };
        let pg_query::protobuf::node::Node::Constraint(c) = node else {
            return Err(ParseError::Structural {
                location: location.clone(),
                message: format!("CREATE DOMAIN {qname}: unexpected constraint node"),
            });
        };
        match ConstrType::try_from(c.contype).unwrap_or(ConstrType::ConstrNull) {
            ConstrType::ConstrNotnull => {
                nullable = false;
            }
            ConstrType::ConstrDefault => {
                let raw = c.raw_expr.as_deref().ok_or_else(|| ParseError::Structural {
                    location: location.clone(),
                    message: format!("CREATE DOMAIN {qname}: DEFAULT without expression"),
                })?;
                default = Some(NormalizedExpr::from_node(raw).map_err(|e| ParseError::Structural {
                    location: location.clone(),
                    message: format!("CREATE DOMAIN {qname}: cannot canonicalize DEFAULT — {e}"),
                })?);
            }
            ConstrType::ConstrCheck => {
                let name = if c.conname.is_empty() {
                    // Auto-generate a synthetic name; PG itself generates one
                    // like `<typname>_check`. Pgevolve normalizes by always
                    // requiring an explicit name in source; otherwise we can't
                    // diff stable. Mirror v0.1 table-check behavior.
                    return Err(ParseError::Structural {
                        location: location.clone(),
                        message: format!(
                            "CREATE DOMAIN {qname}: CHECK constraints must be explicitly named (use CONSTRAINT <name> CHECK (...))"
                        ),
                    });
                } else {
                    Identifier::from_unquoted(&c.conname).map_err(|e| ParseError::Structural {
                        location: location.clone(),
                        message: format!("CREATE DOMAIN {qname}: invalid CHECK name {:?} — {e}", c.conname),
                    })?
                };
                if !seen_check_names.insert(name.as_str().to_string()) {
                    return Err(ParseError::Structural {
                        location: location.clone(),
                        message: format!(
                            "CREATE DOMAIN {qname}: duplicate CHECK name {:?}",
                            name.as_str()
                        ),
                    });
                }
                let raw = c.raw_expr.as_deref().ok_or_else(|| ParseError::Structural {
                    location: location.clone(),
                    message: format!("CREATE DOMAIN {qname}: CHECK without expression"),
                })?;
                let expression =
                    NormalizedExpr::from_node(raw).map_err(|e| ParseError::Structural {
                        location: location.clone(),
                        message: format!("CREATE DOMAIN {qname}: cannot canonicalize CHECK — {e}"),
                    })?;
                checks.push(DomainCheck { name, expression });
            }
            other => {
                return Err(ParseError::Structural {
                    location: location.clone(),
                    message: format!(
                        "CREATE DOMAIN {qname}: unsupported constraint type {other:?} (v0.2 supports NOT NULL, DEFAULT, CHECK)"
                    ),
                });
            }
        }
    }

    // Collation: stmt.coll_clause carries CollateClause; v0.2 ignores it for
    // domains (rare in practice). Document but don't fail.
    let collation: Option<QualifiedName> = None;

    let _ = file;
    Ok(UserType {
        qname,
        kind: UserTypeKind::Domain {
            base,
            nullable,
            default,
            check_constraints: checks,
            collation,
        },
        comment: None,
    })
}
```

```rust
//! Source-side parser for `CREATE TYPE x AS (...)` (composite types).

use pg_query::protobuf::CompositeTypeStmt;

use crate::identifier::{Identifier, QualifiedName};
use crate::ir::user_type::{CompositeAttribute, UserType, UserTypeKind};
use crate::parse::builder::shared::{
    column_type_from_type_name, qname_from_qualified_list,
};
use crate::parse::error::{ParseError, SourceLocation};

pub(crate) fn build_composite(
    stmt: &CompositeTypeStmt,
    file: &std::path::Path,
    location: SourceLocation,
) -> Result<UserType, ParseError> {
    // stmt.typevar is a RangeVar identifying the composite's qualified name.
    let typevar = stmt.typevar.as_ref().ok_or_else(|| ParseError::Structural {
        location: location.clone(),
        message: "CREATE TYPE ... AS: missing target name".into(),
    })?;
    let qname = QualifiedName::new(
        if typevar.schemaname.is_empty() {
            // Will be qualified by the @pgevolve schema directive at a higher
            // level; for now, leave the schema part empty and let the
            // dispatcher resolve via the schema directive. Look at how the
            // view builder handles this.
            return Err(ParseError::Structural {
                location: location.clone(),
                message: "CREATE TYPE ... AS: type must be schema-qualified".into(),
            });
        } else {
            Identifier::from_unquoted(&typevar.schemaname).map_err(|e| ParseError::Structural {
                location: location.clone(),
                message: format!("invalid schema: {e}"),
            })?
        },
        Identifier::from_unquoted(&typevar.relname).map_err(|e| ParseError::Structural {
            location: location.clone(),
            message: format!("invalid type name: {e}"),
        })?,
    );

    // stmt.coldeflist is a list of ColumnDef nodes (one per attribute).
    if stmt.coldeflist.is_empty() {
        return Err(ParseError::Structural {
            location: location.clone(),
            message: format!("CREATE TYPE {qname} AS (...) requires at least one attribute"),
        });
    }

    let mut attributes: Vec<CompositeAttribute> = Vec::with_capacity(stmt.coldeflist.len());
    let mut seen = std::collections::BTreeSet::new();
    for node in &stmt.coldeflist {
        let Some(inner) = &node.node else { continue };
        let pg_query::protobuf::node::Node::ColumnDef(cd) = inner else {
            return Err(ParseError::Structural {
                location: location.clone(),
                message: format!("CREATE TYPE {qname}: unexpected attribute node"),
            });
        };
        let name = Identifier::from_unquoted(&cd.colname).map_err(|e| ParseError::Structural {
            location: location.clone(),
            message: format!("CREATE TYPE {qname}: invalid attribute name {:?} — {e}", cd.colname),
        })?;
        if !seen.insert(name.as_str().to_string()) {
            return Err(ParseError::Structural {
                location: location.clone(),
                message: format!(
                    "CREATE TYPE {qname}: duplicate attribute name {:?}",
                    name.as_str()
                ),
            });
        }
        let type_name = cd.type_name.as_ref().ok_or_else(|| ParseError::Structural {
            location: location.clone(),
            message: format!("CREATE TYPE {qname}: attribute {} missing type", name.as_str()),
        })?;
        let ty = column_type_from_type_name(type_name).map_err(|e| ParseError::Structural {
            location: location.clone(),
            message: format!(
                "CREATE TYPE {qname}: attribute {} has unsupported type — {e}",
                name.as_str()
            ),
        })?;
        // Collations on composite attributes are rare; v0.2 ignores them
        // (set to None). Future sub-spec can extend if needed.
        attributes.push(CompositeAttribute { name, ty, collation: None });
    }

    let _ = file;
    Ok(UserType {
        qname,
        kind: UserTypeKind::Composite { attributes },
        comment: None,
    })
}
```

```rust
fn resolve_user_defined_references(
    catalog: &Catalog,
    locations: &BTreeMap<String, SourceLocation>,
    errors: &mut Vec<AstResolutionError>,
) {
    use crate::ir::column_type::ColumnType;
    use crate::ir::user_type::UserTypeKind;
    use std::collections::BTreeSet;

    let known: BTreeSet<_> = catalog.types.iter().map(|t| t.qname.clone()).collect();

    let mut check_type = |referer: &str, qname: &crate::identifier::QualifiedName, errors: &mut Vec<AstResolutionError>| {
        if !known.contains(qname) {
            errors.push(AstResolutionError {
                message: format!(
                    "{referer} references type {qname} which is not declared in source"
                ),
                location: locations.get(referer).cloned(),
            });
        }
    };

    // Table columns
    for table in &catalog.tables {
        for column in &table.columns {
            if let ColumnType::UserDefined(qname) = &column.ty {
                let loc_key = format!("tables.{}.columns.{}", table.qname, column.name);
                check_type(&loc_key, qname, errors);
            }
        }
    }

    // Composite attributes + Domain bases
    for ut in &catalog.types {
        match &ut.kind {
            UserTypeKind::Composite { attributes } => {
                for attr in attributes {
                    if let ColumnType::UserDefined(qname) = &attr.ty {
                        let loc_key = format!("types.{}.attributes.{}", ut.qname, attr.name);
                        check_type(&loc_key, qname, errors);
                    }
                }
            }
            UserTypeKind::Domain { base, .. } => {
                if let ColumnType::UserDefined(qname) = base {
                    let loc_key = format!("types.{}.base", ut.qname);
                    check_type(&loc_key, qname, errors);
                }
            }
            UserTypeKind::Enum { .. } => {}
        }
    }
}
```

```rust
//! Diff user-defined types (enums, domains, composites).

use std::collections::BTreeMap;

use crate::diff::change::{Change, UserTypeChange};
use crate::identifier::QualifiedName;
use crate::ir::user_type::{CompositeAttribute, EnumValue, UserType, UserTypeKind};

pub fn diff_user_types(catalog: &[UserType], source: &[UserType]) -> Vec<Change> {
    let cat: BTreeMap<_, _> = catalog.iter().map(|t| (t.qname.clone(), t)).collect();
    let src: BTreeMap<_, _> = source.iter().map(|t| (t.qname.clone(), t)).collect();

    let all_keys: std::collections::BTreeSet<_> = cat.keys().chain(src.keys()).cloned().collect();
    let mut out = Vec::new();

    for key in all_keys {
        match (cat.get(&key), src.get(&key)) {
            (None, Some(s)) => out.push(Change::UserType(UserTypeChange::Create((*s).clone()))),
            (Some(_c), None) => out.push(Change::UserType(UserTypeChange::Drop(key.clone()))),
            (Some(c), Some(s)) => diff_same_qname(c, s, &mut out),
            (None, None) => unreachable!(),
        }
    }
    out
}

fn diff_same_qname(catalog: &UserType, source: &UserType, out: &mut Vec<Change>) {
    // Comment delta (always safe).
    if catalog.comment != source.comment {
        out.push(Change::UserType(UserTypeChange::SetComment {
            qname: source.qname.clone(),
            comment: source.comment.clone(),
        }));
    }

    // Kind mismatch → ReplaceWithCascade.
    match (&catalog.kind, &source.kind) {
        (UserTypeKind::Enum { values: c_vals }, UserTypeKind::Enum { values: s_vals }) => {
            diff_enum(&source.qname, c_vals, s_vals, out, catalog, source);
        }
        (UserTypeKind::Domain { .. }, UserTypeKind::Domain { .. }) => {
            diff_domain(catalog, source, out);
        }
        (UserTypeKind::Composite { attributes: c_attrs }, UserTypeKind::Composite { attributes: s_attrs }) => {
            diff_composite(&source.qname, c_attrs, s_attrs, out, catalog, source);
        }
        _ => {
            // Kind changed entirely → recreate.
            out.push(Change::UserType(UserTypeChange::ReplaceWithCascade {
                source: source.clone(),
                catalog: catalog.clone(),
            }));
        }
    }
}

fn diff_enum(
    qname: &QualifiedName,
    catalog: &[EnumValue],
    source: &[EnumValue],
    out: &mut Vec<Change>,
    catalog_ut: &UserType,
    source_ut: &UserType,
) {
    if !enum_can_alter_in_place(catalog, source) {
        out.push(Change::UserType(UserTypeChange::ReplaceWithCascade {
            source: source_ut.clone(),
            catalog: catalog_ut.clone(),
        }));
        return;
    }

    // Emit ADD VALUE for each new value, placed BEFORE the value that follows
    // it in source order (or AFTER the previous value if at end).
    let catalog_names: std::collections::BTreeSet<_> =
        catalog.iter().map(|v| v.name.clone()).collect();
    for (idx, v) in source.iter().enumerate() {
        if catalog_names.contains(&v.name) {
            continue;
        }
        // New value. Determine positioning.
        let (before, after) = if idx + 1 < source.len() && catalog_names.contains(&source[idx + 1].name) {
            (Some(source[idx + 1].name.clone()), None)
        } else if idx > 0 && catalog_names.contains(&source[idx - 1].name) {
            (None, Some(source[idx - 1].name.clone()))
        } else {
            // First value or surrounded by other new values; place at end.
            (None, None)
        };
        out.push(Change::UserType(UserTypeChange::EnumAddValue {
            qname: qname.clone(),
            value: v.name.clone(),
            before,
            after,
        }));
    }

    // Rename detection: position-pairing. A value present in catalog but
    // not source at the same index AS a value present in source but not
    // catalog → rename.
    // For v0.2 we use a conservative heuristic: if both lists have the same
    // length, compare position-by-position; same-position mismatches with
    // exclusive name presence are renames. Otherwise skip (would require
    // ReplaceWithCascade per the compatibility predicate, but we already
    // returned early).
    if catalog.len() == source.len() {
        for (cv, sv) in catalog.iter().zip(source.iter()) {
            if cv.name != sv.name {
                // Both names must be position-paired exclusively.
                let source_has_cat = source.iter().any(|v| v.name == cv.name);
                let catalog_has_src = catalog.iter().any(|v| v.name == sv.name);
                if !source_has_cat && !catalog_has_src {
                    out.push(Change::UserType(UserTypeChange::EnumRenameValue {
                        qname: qname.clone(),
                        from: cv.name.clone(),
                        to: sv.name.clone(),
                    }));
                }
            }
        }
    }
}

pub(crate) fn enum_can_alter_in_place(catalog: &[EnumValue], source: &[EnumValue]) -> bool {
    // For every value present on both sides (matched by name), relative
    // order in source must match relative order in catalog.
    let common: Vec<&str> = source
        .iter()
        .filter(|s| catalog.iter().any(|c| c.name == s.name))
        .map(|v| v.name.as_str())
        .collect();
    let catalog_order: Vec<&str> = catalog
        .iter()
        .filter(|c| source.iter().any(|s| s.name == c.name))
        .map(|v| v.name.as_str())
        .collect();
    // If common-values' relative order differs, in-place is impossible.
    if common != catalog_order {
        // Could still be salvageable if it's a pure rename (same-length lists
        // with position-paired exclusive names) — but that's handled by
        // diff_enum's rename detection AFTER this returns true. Reject only
        // when there's a real reordering of preserved values.
        return false;
    }
    // Also reject if catalog has values not in source AND it's not a rename
    // case (different length OR more than position-pair renames).
    let dropped: Vec<&str> = catalog
        .iter()
        .filter(|c| !source.iter().any(|s| s.name == c.name))
        .map(|v| v.name.as_str())
        .collect();
    if !dropped.is_empty() {
        // Drops are only OK if they pair 1:1 with adds at the same indexes
        // (rename heuristic). If lengths differ OR drop count > add count,
        // this is a true drop and we can't ALTER.
        let added: Vec<&str> = source
            .iter()
            .filter(|s| !catalog.iter().any(|c| c.name == s.name))
            .map(|v| v.name.as_str())
            .collect();
        if catalog.len() != source.len() || dropped.len() != added.len() {
            return false;
        }
        // Same length + same drop/add counts → could be all-renames. The
        // position-pairing check in diff_enum handles the actual matching.
    }
    true
}

fn diff_domain(catalog: &UserType, source: &UserType, out: &mut Vec<Change>) {
    let UserTypeKind::Domain {
        base: c_base, nullable: c_null, default: c_default,
        check_constraints: c_checks, ..
    } = &catalog.kind else { unreachable!() };
    let UserTypeKind::Domain {
        base: s_base, nullable: s_null, default: s_default,
        check_constraints: s_checks, ..
    } = &source.kind else { unreachable!() };

    // Base type change → ReplaceWithCascade (PG can't ALTER a domain's base).
    if c_base != s_base {
        out.push(Change::UserType(UserTypeChange::ReplaceWithCascade {
            source: source.clone(),
            catalog: catalog.clone(),
        }));
        return;
    }

    if c_null != s_null {
        out.push(Change::UserType(UserTypeChange::DomainSetNotNull {
            qname: source.qname.clone(),
            not_null: !s_null,
        }));
    }

    if c_default != s_default {
        out.push(Change::UserType(UserTypeChange::DomainSetDefault {
            qname: source.qname.clone(),
            default: s_default.clone(),
        }));
    }

    // CHECK constraints: pair by name.
    let c_map: BTreeMap<_, _> = c_checks.iter().map(|c| (&c.name, c)).collect();
    let s_map: BTreeMap<_, _> = s_checks.iter().map(|c| (&c.name, c)).collect();
    for (name, sc) in &s_map {
        match c_map.get(name) {
            None => out.push(Change::UserType(UserTypeChange::DomainAddCheck {
                qname: source.qname.clone(),
                constraint: (*sc).clone(),
            })),
            Some(cc) if cc.expression != sc.expression => {
                // Replace: drop + add.
                out.push(Change::UserType(UserTypeChange::DomainDropCheck {
                    qname: source.qname.clone(),
                    name: (*name).clone(),
                }));
                out.push(Change::UserType(UserTypeChange::DomainAddCheck {
                    qname: source.qname.clone(),
                    constraint: (*sc).clone(),
                }));
            }
            Some(_) => {} // unchanged
        }
    }
    for (name, _) in &c_map {
        if !s_map.contains_key(name) {
            out.push(Change::UserType(UserTypeChange::DomainDropCheck {
                qname: source.qname.clone(),
                name: (*name).clone(),
            }));
        }
    }
}

fn diff_composite(
    qname: &QualifiedName,
    catalog: &[CompositeAttribute],
    source: &[CompositeAttribute],
    out: &mut Vec<Change>,
    catalog_ut: &UserType,
    source_ut: &UserType,
) {
    if !composite_can_alter_in_place(catalog, source) {
        out.push(Change::UserType(UserTypeChange::ReplaceWithCascade {
            source: source_ut.clone(),
            catalog: catalog_ut.clone(),
        }));
        return;
    }

    let catalog_names: std::collections::BTreeSet<_> =
        catalog.iter().map(|a| a.name.clone()).collect();
    let source_names: std::collections::BTreeSet<_> =
        source.iter().map(|a| a.name.clone()).collect();

    // Drops first.
    for c_attr in catalog {
        if !source_names.contains(&c_attr.name) {
            out.push(Change::UserType(UserTypeChange::CompositeDropAttribute {
                qname: qname.clone(),
                name: c_attr.name.clone(),
            }));
        }
    }

    // Type changes for attributes present on both sides.
    for s_attr in source {
        if let Some(c_attr) = catalog.iter().find(|c| c.name == s_attr.name) {
            if c_attr.ty != s_attr.ty {
                out.push(Change::UserType(UserTypeChange::CompositeAlterAttributeType {
                    qname: qname.clone(),
                    attribute: s_attr.name.clone(),
                    new_type: s_attr.ty.clone(),
                }));
            }
        }
    }

    // Adds last.
    for s_attr in source {
        if !catalog_names.contains(&s_attr.name) {
            out.push(Change::UserType(UserTypeChange::CompositeAddAttribute {
                qname: qname.clone(),
                attribute: s_attr.clone(),
            }));
        }
    }
}

pub(crate) fn composite_can_alter_in_place(
    catalog: &[CompositeAttribute],
    source: &[CompositeAttribute],
) -> bool {
    // Reject reorderings: relative order of attributes present on both
    // sides must match.
    let common_in_catalog: Vec<&str> = catalog
        .iter()
        .filter(|c| source.iter().any(|s| s.name == c.name))
        .map(|a| a.name.as_str())
        .collect();
    let common_in_source: Vec<&str> = source
        .iter()
        .filter(|s| catalog.iter().any(|c| c.name == s.name))
        .map(|a| a.name.as_str())
        .collect();
    common_in_catalog == common_in_source
}
```

```rust
//! SQL emission for user-defined type planner steps.

use crate::identifier::{Identifier, QualifiedName};
use crate::ir::column_type::ColumnType;
use crate::ir::user_type::{
    CompositeAttribute, DomainCheck, EnumValue, UserType, UserTypeKind,
};
use crate::parse::normalize_expr::NormalizedExpr;

pub(crate) fn emit_create_type(ut: &UserType) -> String {
    match &ut.kind {
        UserTypeKind::Enum { values } => emit_create_enum(&ut.qname, values),
        UserTypeKind::Domain { base, nullable, default, check_constraints, .. } => {
            emit_create_domain(&ut.qname, base, *nullable, default.as_ref(), check_constraints)
        }
        UserTypeKind::Composite { attributes } => emit_create_composite(&ut.qname, attributes),
    }
}

fn emit_create_enum(qname: &QualifiedName, values: &[EnumValue]) -> String {
    let mut s = format!("CREATE TYPE {qname} AS ENUM (");
    for (i, v) in values.iter().enumerate() {
        if i > 0 { s.push_str(", "); }
        s.push('\'');
        s.push_str(&v.name.replace('\'', "''"));
        s.push('\'');
    }
    s.push_str(");");
    s
}

fn emit_create_domain(
    qname: &QualifiedName,
    base: &ColumnType,
    nullable: bool,
    default: Option<&NormalizedExpr>,
    checks: &[DomainCheck],
) -> String {
    let mut s = format!("CREATE DOMAIN {qname} AS {}", base.render_sql());
    if !nullable {
        s.push_str(" NOT NULL");
    }
    if let Some(d) = default {
        s.push_str(" DEFAULT ");
        s.push_str(d.canonical_text());
    }
    for c in checks {
        s.push_str(&format!(" CONSTRAINT {} CHECK ({})", c.name.as_str(), c.expression.canonical_text()));
    }
    s.push(';');
    s
}

fn emit_create_composite(qname: &QualifiedName, attrs: &[CompositeAttribute]) -> String {
    let mut s = format!("CREATE TYPE {qname} AS (\n");
    for (i, a) in attrs.iter().enumerate() {
        if i > 0 { s.push_str(",\n"); }
        s.push_str("    ");
        s.push_str(a.name.as_str());
        s.push(' ');
        s.push_str(&a.ty.render_sql());
    }
    s.push_str("\n);");
    s
}

pub(crate) fn emit_drop_type(qname: &QualifiedName) -> String {
    format!("DROP TYPE {qname};")
}

pub(crate) fn emit_alter_type_add_value(
    qname: &QualifiedName,
    value: &str,
    before: Option<&str>,
    after: Option<&str>,
) -> String {
    let escaped = value.replace('\'', "''");
    let mut s = format!("ALTER TYPE {qname} ADD VALUE '{escaped}'");
    if let Some(b) = before {
        s.push_str(&format!(" BEFORE '{}'", b.replace('\'', "''")));
    } else if let Some(a) = after {
        s.push_str(&format!(" AFTER '{}'", a.replace('\'', "''")));
    }
    s.push(';');
    s
}

pub(crate) fn emit_alter_type_rename_value(qname: &QualifiedName, from: &str, to: &str) -> String {
    format!(

        to.replace('\'', "''"),
    )
}
```

```rust
fn type_shadows_table_rule(catalog: &Catalog, out: &mut Vec<Finding>) {
    let relation_qnames: std::collections::BTreeSet<_> = catalog.tables.iter().map(|t| t.qname.clone())
        .chain(catalog.views.iter().map(|v| v.qname.clone()))
        .chain(catalog.materialized_views.iter().map(|m| m.qname.clone()))
        .collect();
    for t in &catalog.types {
        if relation_qnames.contains(&t.qname) {
            out.push(Finding::error(
                "type-shadows-table",
                format!(
                    "user-defined type {} collides with a table/view/MV of the same qualified name",
                    t.qname,
                ),
            ));
        }
    }
}

fn enum_value_collision_rule(catalog: &Catalog, out: &mut Vec<Finding>) {
    use crate::ir::user_type::UserTypeKind;
    for t in &catalog.types {
        if let UserTypeKind::Enum { values } = &t.kind {
            let mut seen = std::collections::BTreeSet::new();
            for v in values {
                if !seen.insert(v.name.clone()) {
                    out.push(Finding::error(
                        "enum-value-collision",
                        format!("enum {} has duplicate value {:?}", t.qname, v.name),
                    ));
                }
            }
        }
    }
}

fn composite_attribute_collision_rule(catalog: &Catalog, out: &mut Vec<Finding>) {
    use crate::ir::user_type::UserTypeKind;
    for t in &catalog.types {
        if let UserTypeKind::Composite { attributes } = &t.kind {
            let mut seen = std::collections::BTreeSet::new();
            for a in attributes {
                if !seen.insert(a.name.as_str().to_string()) {
                    out.push(Finding::error(
                        "composite-attribute-collision",
                        format!("composite {} has duplicate attribute {:?}", t.qname, a.name.as_str()),
                    ));
                }
            }
        }
    }
}

fn domain_check_references_unmanaged_type_rule(
    catalog: &Catalog,
    managed: &ManagedConfig,
    out: &mut Vec<Finding>,
) {
    use crate::ir::user_type::UserTypeKind;
    if managed.schemas.is_empty() { return; }
    let managed_schemas: std::collections::BTreeSet<_> =
        managed.schemas.iter().map(|s| s.as_str().to_string()).collect();
    const BUILTINS: &[&str] = &["pg_catalog", "information_schema"];

    for t in &catalog.types {
        if let UserTypeKind::Domain { check_constraints, .. } = &t.kind {
            for check in check_constraints {
                // Inspect the check expression's dependencies (if NormalizedExpr
                // tracks them) — for v0.2 a simple text-scan over identifiers
                // is enough to catch the common case. Skip if no dependency
                // tracking is available; the lint exists primarily for the
                // forward-looking case where Functions land.
                let text = check.expression.canonical_text();
                // Heuristic: if the text contains "schema.foo" where schema
                // is not in managed_schemas or builtins, warn.
                for (schema, _name) in extract_qualified_refs(text) {
                    if !managed_schemas.contains(&schema) && !BUILTINS.contains(&schema.as_str()) {
                        out.push(Finding::warning(
                            "domain-check-references-unmanaged-type",
                            format!(
                                "domain {} CHECK references unmanaged schema {schema} via expression `{text}`",
                                t.qname,
                            ),
                        ));
                    }
                }
            }
        }
    }
}

// Helper: extract `schema.name` substrings from a SQL expression. Very
// simple regex-free implementation for v0.2.
fn extract_qualified_refs(text: &str) -> Vec<(String, String)> {
    let mut out = Vec::new();
    // Tokenize on whitespace and punctuation; look for tokens containing `.`.
    for token in text.split(|c: char| {
        !(c.is_alphanumeric() || c == '_' || c == '.')
    }) {
        if let Some((schema, name)) = token.split_once('.') {
            if !schema.is_empty() && !name.is_empty()
                && schema.chars().all(|c| c.is_alphanumeric() || c == '_')
                && name.chars().all(|c| c.is_alphanumeric() || c == '_')
            {
                out.push((schema.to_string(), name.to_string()));
            }
        }
    }
    out
}
```
